mruby-pure-regexp
=================

Pure mruby Regexp

[![Build Status](https://travis-ci.org/h2so5/mruby-pure-regexp.svg?branch=master)](https://travis-ci.org/h2so5/mruby-pure-regexp)

## Metacharacters

* ```.``` Any single character
* ```?``` Zero or one
* ```^``` Start of line
* ```$``` End of line
* ```()``` Group
* ```(?:)``` Non-capturing group

## Options

* ```/i``` Case insensitive
* ```/m``` Multiline mode
