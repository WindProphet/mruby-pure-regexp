mruby-pure-regexp
=================

Pure mruby Regexp

[![Build Status](https://travis-ci.org/h2so5/mruby-pure-regexp.svg?branch=master)](https://travis-ci.org/h2so5/mruby-pure-regexp)

## Metacharacters

* ```()``` Group
* ```.``` Any single character
* ```?``` Zero or more
* ```^``` Start of line
* ```$``` End of line

## Options

* ```/i``` Case insensitive
* ```/m``` Multiline mode
